How to use
If you want to use this app, you need to enter an accurate prompt in prompt input and chose gpt model. Here is an example: Дай мне вопрос для начинающего в изучении python, не говори ничего лишнего, просто вопрос. Твой ответ должен содержать не более 12 слов. Then you need to press "press" button, task will appear. After you should enter your answer to the question in "Your answer" field and press "Check" button.
